## 👨‍💻 About Me

### DevOps & Cloud Engineer

I'm a DevOps & Cloud Engineer based in New Cairo, Egypt. I recently completed a hands-on DevOps internship at the Information Technology Institute (ITI), where I built real-world CI/CD pipelines, managed containerized workloads, and provisioned cloud infrastructure using IaC. My focus is on cloud-native infrastructure, GitOps, and automation — with a goal to automate and scale everything, one pipeline at a time. I'm actively looking for **Junior DevOps / Cloud Engineer opportunities** where I can contribute, grow, and keep pushing the boundaries of what automation can do.
